## 📊 Project Overview
The goal of this project is to build a predictive model to find out the sales of each product at a particular store. This helps BigMart understand the properties of products and stores which play a crucial role in increasing sales.

## 🛠️ Tools & Libraries Used
* **Python** (Data Analysis)
* **Pandas & NumPy** (Data Cleaning)
* **Matplotlib & Seaborn** (EDA)
* **Scikit-Learn** (Model Building)
* **XGBoost** (Advanced Modeling)

## 📈 Key Insights
* **Item_MRP** is the strongest driver of sales.
* **Supermarket Type 3** stores have the highest sales volume.
* **Store Age** (Outlet_Age) significantly impacts customer trust and sales stability.

## 🏆 Best Model Results
* **Winning Model:** Random Forest / XGBoost
* **R2 Score:** 0.61 (Approx)
* **RMSE:** 1080 (Approx)
